electric_bolt

電工デイリー・インサイト

AI要約に基づく電気工事の最新トレンドと影響度分析

terminal 最新ニュースを手動で収集・更新する方法

バックエンドの通信スクリプトを実行することで、指定のメーカー（パナソニック・三菱電機・ダイキン）のサイトから最新のプレスリリース情報を取得し、表示を更新します。

# 初回のみ必要なコマンド
npm install

# ニュースの最新化コマンド
node scraper.js

info スクリプトで「収集完了」と表示された後、このページをブラウザでリロード（再読み込み）すると更新が反映されます。
